url_shortcut

Стек технологий: java 8, SpringBoot, MySQL/PostgreSQL(в зависимости от выбанного вами файла), liquibase, авторизация через jwt, Добавлено логирование с ипользованием Aspect, Написаны тесты junit/mockito, фронт Angular.

Для входа в приложение можно использовать admin/admin или user/user также можно зарегистрироваться указав домен

    Приложение состоит из двух частей:

  • Backend https://github.com/AlexandrKaleganov/url_shortcut_back
  • Frontend https://github.com/AlexandrKaleganov/url_shortcut_frontend

В данном приложении реализован следующий функционал:

  1. Пользователь может зарегистрироваться, чтобы зарегистрироваться необходимо указать домен, например: 'google.com' без http, www и всего лишнего.
  2. После регистрации пользователь может добавлять длинные ссылки.
  3. В форму уже подставляется домен автоматически.
  4. На вкладке "ссылки" мы можем наблюдать список ссылок и коротких ссылок к ним.
  5. Список ссылок, Список пользователей, Статистика отображается с пагинацией.
  6. Пользователей можно фильтровать по домену, по логину, по фамилии, Ссылки можно фильтровать по логину пользователя.
